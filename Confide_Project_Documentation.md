## 1. КОНЦЕПЦИЯ И ПОЗИЦИОНИРОВАНИЕ

### Суть продукта

**Confide** — платформа психологической поддержки нового поколения.

**НЕ** AI-психолог. **НЕ** медицинский сервис. **НЕ** чат-бот.

**А** — умный собеседник по имени Алекс (или любое имя которое выберет пользователь), который помнит всю историю разговоров, адаптируется под стиль общения и с каждой сессией понимает пользователя глубже.

> **Позиционирование:** "Разговор по душам — место где тебя всегда выслушают"

### Почему "Confide"

"To confide" — доверять кому-то сокровенное, делиться тем что не скажешь другим. Точно описывает суть продукта. Работает на обоих рынках: EN и RU.

### Ключевые принципы

- **НЕ психолог, НЕ терапия** → wellness & self-help платформа (снимает регуляторные риски)
- **НЕ чат-бот** → живые отношения которые эволюционируют месяцами
- **НЕ стоковые ответы** → глубокий анализ через RAG базу знаний
- **Уникальный голос** для каждого пользователя (ElevenLabs Voice Design)
- **Имя собеседника** выбирает сам пользователь при регистрации

### Целевая аудитория

| Рынок | Портрет | Боль |
|-------|---------|------|
| EN (основной) | 25–45 лет, США/Канада/UK/Австралия | Психолог $80–300/час, нет 24/7, нет анонимности |
| RU диаспора | Эмигранты в США/Германии/Израиле | Языковой барьер к местным психологам, платёжеспособность западная |

### Эволюция отношений с агентом

```
Месяц 1:  Осторожно, формально — "Расскажи подробнее..."
Месяц 3:  Теплее — "Ты говорил про маму — как там?"
Месяц 6:  Как старый друг — "Ты снова так говоришь когда устал"
Год+:     "За год ты заметно спокойнее — ты сам это замечаешь?"
```

## 2. АНАЛИЗ КОНКУРЕНТОВ И НАШИ УТП

### Главные конкуренты

**Wysa** — CBT пингвин
- Цена: $75/год
- Минусы: Нет долгосрочной памяти, сбрасывает контекст после сессии
- Плюсы: FDA approved, дыхательные упражнения, marketplace живых терапевтов

**Woebot** — CBT чат-бот
- Цена: Бесплатно (FDA clearance)
- Минусы: Сбрасывает контекст, только текст, нет голоса
- Плюсы: Проверенные CBT/DBT техники, утренние check-ins

**Replika** — 3D аватар-компаньон
- Цена: $19/мес
- Минусы: Нет психологической базы знаний, развлечение не помощь
- Плюсы: 3D аватар, долгосрочная память, voice

**Youper** — mood tracker + AI
- Цена: $89/год
- Минусы: Мало глубины, поверхностные советы
- Плюсы: Mood tracking с графиками

**BetterHelp** — живые терапевты
- Цена: $60-90/неделю ($240-360/мес!)
- Минусы: Дорого, нет 24/7, нет анонимности
- Плюсы: Реальные лицензированные терапевты

### Наши УТП (чего нет ни у кого)

1. **Долгосрочная живая память** — помнит всё с первого дня, эволюция отношений
2. **RAG база знаний** — отвечает на основе реальных психологических методик (Wysa/Woebot так не делают)
3. **Уникальный голос для каждого** — не стоковый TTS (только у Replika есть голос)
4. **Специализированные агенты** — не один универсальный бот
5. **Цена $19/мес** — в 10 раз дешевле BetterHelp, доступнее Replika
6. **Crisis detection с первого дня** — безопасность (Wysa умеет, остальные нет)

### Что возьмём у конкурентов

От **Wysa**:
- Дыхательные упражнения как отдельные модули
- Marketplace живых психологов (v2.0)

От **Woebot**:
- Утренние check-ins "как ты сегодня?"
- Mood score 1-10 перед началом сессии

От **Youper**:
- График настроения с визуализацией

От **Replika**:
- 3D аватар собеседника опционально (v2.0+)

## 3. ПОЛНЫЙ СПИСОК ФИЧ

### ✅ MVP — Версия 0.1 (делаем первыми)

**Аутентификация и кабинет**
- Регистрация через email / Google OAuth
- Личный кабинет пользователя
- Онбординг с дисклеймером (не медицинский сервис — юридическая защита)
- Квиз при регистрации: выбор пола, возраста, тембра голоса собеседника
- Присвоение имени собеседнику (Алекс, Анна — или своё)

**Текстовый чат с агентом**
- Чат-интерфейс с историей сообщений
- Индикатор печати "Алекс думает..."
- Показ источников из RAG — какие книги/методики использованы
- Кнопка "сохранить инсайт" для личного дневника

**Система памяти (ключевая фича)**
- Краткосрочная память: контекст текущего разговора (last 30 сообщений)
- Долгосрочная память: summary каждой сессии (LLM сжимает в 5–7 фактов)
- Профиль пользователя — обновляется автоматически после каждой сессии
- Векторный поиск по истории разговоров (семантический, Pinecone)
- Агент запоминает: ключевых людей, триггеры, паттерны, прогресс

**RAG база знаний**
- Загрузка психологической литературы в векторную БД
- Сегментированные namespace по специализациям
- Агент отвечает на основе реальных методик

**Crisis Detection Layer (обязателен с первого дня)**
- Отдельный параллельный агент анализирует каждое сообщение
- При кризисных триггерах — немедленное переключение на протокол безопасности
- Вывод контактов кризисных линий по стране пользователя
- Жёстко захардкожено — LLM не принимает решений в кризисе

**Монетизация**
- Freemium: 5 сессий бесплатно → платная подписка
- Stripe subscriptions: $19/мес и $29/мес

## 4. ТЕХНИЧЕСКИЙ СТЕК И МОБИЛЬНАЯ СТРАТЕГИЯ

| Слой | Технология | Почему |

| Стили | Tailwind CSS + shadcn/ui | Быстрый профессиональный UI |
| Анимации | Framer Motion | Плавные переходы |
| State | Zustand | Простой глобальный state |
| БД + Auth | Supabase | PostgreSQL + Auth + Storage + Realtime в одном |
| ORM | Prisma | Типобезопасная работа с БД |
| LLM | GPT-4o (OpenAI API) | Основная модель агентов |
| Агенты | LangChain.js | Оркестрация, цепочки, память |
| Векторная БД | Pinecone | RAG — хранение базы знаний |
| STT | Whisper API (OpenAI) | Голос → текст |
| TTS | ElevenLabs API | Текст → уникальный голос |
| Хостинг | Vercel | Деплой через git push |
| Платежи | Paddle (NOT Stripe) | Merchant of Record — налоги автоматически |
| Email | Resend | Транзакционные письма |
| Аналитика | PostHog | Retention, funnel, события |
| PWA | next-pwa | Progressive Web App (Phase 2) |
| Mobile | React Native | iOS + Android (v2.0+) |

### AI слой — архитектура агентов

```
Orchestrator Agent
├── Определяет специализацию (anxiety / family / trauma / etc)
├── Детектирует кризисность
└── Маршрутизирует запрос

Specialist Agents (×6)
├── anxiety.ts      — CBT, ACT, DBT
├── family.ts       — Gottman, Satir
├── trauma.ts       — van der Kolk, Bowlby
├── relationships.ts — Attachment Theory
├── mens.ts         — Gender-adapted CBT
└── womens.ts       — Gender-adapted CBT

Memory Agent
└── Запускается ПОСЛЕ каждой сессии
    Обновляет user_profile в Supabase

Crisis Agent (параллельный — ВСЕГДА активен)
└── HARDCODED ответ — никогда через LLM
```

### Структура памяти

```
SHORT-TERM:   последние 30 сообщений в контексте
LONG-TERM:    JSON профиль в Supabase
SEMANTIC:     Pinecone — поиск по всей истории
SUMMARY:      GPT-4o сжимает каждую сессию в 5–7 фактов
```

### Мобильная стратегия

**Этап 1: Веб (сейчас)** — Next.js responsive для всех устройств

**Этап 2: PWA (после MVP)** — Progressive Web App
- next-pwa — превращает Next.js в PWA
- Установка на домашний экран (iOS/Android)
- Push уведомления ("Как дела сегодня?")
- Офлайн режим (кэш последних сессий)
- Работает как нативное приложение

**Этап 3: React Native (v2.0+)** — нативные приложения
- Только после доказанного Product-Market Fit
- Переиспользование логики: API, база знаний, промпты
- iOS + Android из одной кодовой базы

**Почему PWA сначала:**
- Одна кодовая база (Next.js)
- Без App Store / Google Play (0% комиссии)
- Обновления мгновенные (не через ревью)
- SEO работает (React Native нет)

### Ключевые промпты для Claude Code

**Старт проекта:**
```
Прочитай CLAUDE.md полностью.
Создай структуру папок из документа.
Установи все зависимости из стека.
Создай .env.example с шаблоном переменных.
Настрой Prisma schema по схеме БД.
```

**RAG система:**
```
Реализуй RAG систему:
PDF загрузка → text chunking (500 токенов) →
OpenAI embeddings → сохранение в Pinecone с metadata
(book_title, chapter, namespace).
Retrieval: top-5 чанков по cosine similarity.
```

**Голосовой компонент:**
```
Создай VoiceChat React компонент:
hold-to-record кнопка через MediaRecorder API →
отправка audio blob на /api/voice →
Whisper транскрипция → агент →
ElevenLabs TTS → воспроизведение.
Статусы: listening / thinking / speaking.
```

**Memory Agent:**
```
После каждой сессии запускай Memory Agent:
анализирует conversation, извлекает 5–7 фактов
о пользователе, обновляет user_profile в Supabase.
Профиль подгружается в system prompt новой сессии.
```

**Crisis Detection:**
```
Создай параллельный CrisisAgent:
анализирует каждое сообщение на триггеры
(суицид, острый кризис, насилие).
При обнаружении — прерывает основной поток,
возвращает HARDCODED ответ с кризисными ресурсами
по стране пользователя.
```

## 8. СХЕМА БАЗЫ ДАННЫХ

```sql
-- Пользователи
users (
  id              UUID PRIMARY KEY,
  email           TEXT UNIQUE NOT NULL,
  created_at      TIMESTAMP DEFAULT NOW(),
  plan            TEXT DEFAULT 'free',      -- free | pro | premium
  voice_id        TEXT,                     -- ElevenLabs voice ID
  companion_name  TEXT DEFAULT 'Alex',
  companion_gender TEXT,                    -- male | female
  language        TEXT DEFAULT 'en'         -- en | ru
)

-- Живой профиль (обновляет Memory Agent после каждой сессии)
user_profiles (
  id                  UUID PRIMARY KEY,
  user_id             UUID REFERENCES users(id) ON DELETE CASCADE,
  communication_style JSONB DEFAULT '{}',
  emotional_profile   JSONB DEFAULT '{}',
  life_context        JSONB DEFAULT '{}',
  patterns            JSONB DEFAULT '[]',
  progress            JSONB DEFAULT '{}',
  what_worked         JSONB DEFAULT '[]',
  updated_at          TIMESTAMP DEFAULT NOW()
)

-- Сессии разговоров
sessions (
  id          UUID PRIMARY KEY,
  user_id     UUID REFERENCES users(id),
  agent_type  TEXT,                        -- anxiety | family | trauma | etc
  created_at  TIMESTAMP DEFAULT NOW(),
  ended_at    TIMESTAMP,
  summary     TEXT,                        -- GPT summary 5–7 фактов
  mood_score  INTEGER                      -- 1–10
)

-- Все сообщения
messages (
  id          UUID PRIMARY KEY,
  session_id  UUID REFERENCES sessions(id),
  user_id     UUID REFERENCES users(id),
  role        TEXT NOT NULL,               -- user | assistant
  content     TEXT NOT NULL,
  audio_url   TEXT,                        -- если голосовое
  created_at  TIMESTAMP DEFAULT NOW(),
  is_crisis   BOOLEAN DEFAULT FALSE
)

-- Личный дневник инсайтов
journal_entries (
  id                UUID PRIMARY KEY,
  user_id           UUID REFERENCES users(id),
  content           TEXT NOT NULL,
  insight           TEXT,
  source_message_id UUID REFERENCES messages(id),
  created_at        TIMESTAMP DEFAULT NOW()
)

-- Stripe подписки
subscriptions (
  id                      UUID PRIMARY KEY,
  user_id                 UUID REFERENCES users(id),
  stripe_customer_id      TEXT UNIQUE,
  stripe_subscription_id  TEXT UNIQUE,
  plan                    TEXT,
  status                  TEXT,            -- active | canceled | past_due
  expires_at              TIMESTAMP
)

-- Метаданные RAG базы знаний
knowledge_base (
  id            UUID PRIMARY KEY,
  source_title  TEXT NOT NULL,
  author        TEXT,
  namespace     TEXT NOT NULL,             -- anxiety_cbt | family | trauma | crisis | general
  chunk_index   INTEGER,
  pinecone_id   TEXT UNIQUE
)
```

## 10. YOUTUBE КАНАЛ

### Концепция

Канал = главный маркетинговый канал платформы. Стоимость привлечения в 10× дешевле платной рекламы.

**Формат одного ролика (15 мин):**
```
0:00  — Ведущие (вступление): диалог Ангел vs Демон, тема серии    [2 мин]
2:00  — История с героями: драматизированная психологическая ситуация [8 мин]
10:00 — Ведущие (разбор): психологические советы по теме истории    [5 мин]
15:00 — CTA: ссылка на Confide, "разбери свою ситуацию с Алексом"
```

| Инструмент | План | Стоимость | Назначение |
|-----------|------|-----------|-----------|
| Higgsfield.ai | Ultimate $39/мес | $39 | Soul ID (персонажи), Kling (видео), Nano Banana (фото 4K) |
| ElevenLabs | Starter | $22 | Голоса ведущих и персонажей |
| HeyGen | Essential | $29 | Lipsync ведущих |
| Suno AI | Pro | $10 | Генерация музыки |
| CapCut Pro | Pro | $10 | Монтаж, субтитры |
| **ИТОГО** | | **$110/мес** | |

### Пайплайн одного ролика

| Шаг | Инструмент | Действие | Время |
|-----|-----------|----------|-------|
| 1 | Claude | Полный скрипт: диалог + история + раскадровка + промпты | 30 мин |
| 2 | Higgsfield Soul ID | Создание персонажей (один раз, переиспользуем) | 1–2 ч (разово) |
| 3 | Higgsfield Nano Banana | 25–30 фото сцен с consistent персонажами | 30–40 мин |

| 6 | ElevenLabs | Озвучка истории — закадровый голос + персонажи | 20 мин |
| 7 | Suno AI | 2–3 трека: фоновый, напряжённый, финальный | 10 мин |
| 8 | CapCut AI | Монтаж: видео + озвучка + музыка + субтитры | 1–1.5 ч |

> **Итого на ролик:** 4–5 часов первые разы, 2–3 часа после отработки пайплайна.

### Стратегия запуска

```
Месяц 1–2:  Первые 4–6 роликов вручную — тест формата
Месяц 3–4:  Находишь что заходит — создаёшь библиотеку персонажей
Месяц 5:    Запуск Confide — канал уже имеет тёплую аудиторию
Месяц 6+:   Автоматизация через n8n
```

### Серии контента

- **"Любовь которая разрушает"** — эмоциональное насилие, токсичные отношения
- **"Родители которые ранят"** — детские травмы, нарциссический родитель
- **"Мужчины не плачут"** — мужская уязвимость, агрессия как маска боли
- **"Тихая тревога"** — панические атаки, генерализованная тревога
- **"Семья на грани"** — развод, кризис отношений, измена
