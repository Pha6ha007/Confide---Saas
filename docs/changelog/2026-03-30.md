# 2026-03-30

## OpenRouter AI Gateway + Multi-Model Architecture

**Полный рефакторинг AI-инфраструктуры.** Перевели весь проект с одного провайдера (Groq/OpenAI) на OpenRouter gateway с автоматическим fallback и моделями по задачам.

### Новая архитектура

- **lib/ai/models.ts** — централизованный реестр всех моделей с fallback chains
- **lib/ai/router.ts** — единый AI клиент с автоматическим переключением при ошибках
